sprintf字符格式化漏洞解析

sprintf函数

sprintf （ string $format [， mixed $... ]）： string
返回根据格式化字符串生成的字符串format

format规则：%[flags][width][.precision]specifier
flag: -、+、空格、0、'(char)[如'^]
width: 一个整数,表示此转换应产生的字符数(最小值)

format特性导致单引号逃逸或丢掉丢掉

  1. '(char)将导致单引号被吞掉
    php > echo sprintf("%'^10s",'od0d');
^^^^^^od0d
  2. 传入字符%1$\',sprintf无法识别会将%1$\字符串扔掉,逃逸出'
php > echo sprintf("%1$\'",'od0d');
'
