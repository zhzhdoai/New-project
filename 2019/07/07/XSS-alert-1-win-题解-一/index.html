Markdown

代码

function escape(s) {
  var text = s.replace(/</g, '&lt;').replace(/"/g, '&quot;');
  // URLs
  text = text.replace(/(http:\/\/\S+)/g, '<a href="$1">$1</a>');
  // [[img123|Description]]
  text = text.replace(/\[\[(\w+)\|(.+?)\]\]/g, '<img alt="$2" src="$1.gif">');
  return text;
}

题解

text变量为对<、"进行html实体编码，在对http://字符串替换为标签a。想到闭合alt标签但是会被"会被实体，发现传入http://引入的"可以闭合alt属性。
