0x01 Ubuntu_LAMP搭建

LAMP命令备忘录：

  1. 更新源：apt-get update
  2. 安装装apche2：apt-get install apache2 -y
  3. 安装mysql:apt-get install mysql_server -y
  4. 安装php5.6:(注意这里就不要执行apt-get install php,会造成麻烦) 安装扩展插件：
  5. 赋予/var/www/html 777权限:chmod 777 /html
  6. 拖入sqlilabs与/var/www/html中

0x02 ubuntu网速优化

  1. 解决网速问题：sudo gedit /etc/modprobe.d/iwlwifi.conf
    文件下写入保存：options iwlwifi 11n_disable=1
  2. 切换跟新源，选择国内
    在setting system中打开（可解决apt-get update 更新出错）

0x03 ubuntu——python环境搭建:
